## Installation

### 1. Clone the repository
```
git clone https://github.com/yosbelm/pediatric_percentile_calculator.git
```
### 2. Install dependencies
```
cd pediatric_percentile_calculator
pip install -r requirements.txt
```

### 3. Run the application
```
python main.py
```
